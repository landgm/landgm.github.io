# 가운데 글자 가져오기


```python
def solution(s):
    answer = ''
    if len(s)%2 == 0:
        a = int(len(s)/2 -1)
        b = int(len(s)/2 +1)
        answer = s[a: b]
    else :
        a = int((len(s)-1)/2)
        answer = s[a]
    
    return answer

```
