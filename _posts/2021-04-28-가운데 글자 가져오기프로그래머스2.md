* 나의 풀이


```python
def solution(s):
    answer = ''
    if len(s)%2 == 0:
        a = int(len(s)/2 -1)
        b = int(len(s)/2 +1)
        answer = s[a: b]
    else :
        a = int((len(s)-1)/2)
        answer = s[a]
    
    return answer

```

def string_middle(str):
    # 함수를 완성하세요
