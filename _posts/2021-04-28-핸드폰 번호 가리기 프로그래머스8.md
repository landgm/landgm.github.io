* replace : 문자열 안에있는 글자를 바꿀 수 있다.


```python
a = '1235'
```


```python
a.replace('3','#')
```

---

* 나의 풀이


```python
def solution(phone_number):
    n = len(phone_number[:-4]) 
    return phone_number.replace(phone_number[:-4] ,'*'*n)
```

* 다른 사람 풀이


```python
def hide_numbers(s):
    return "*"*(len(s)-4) + s[-4:]

# 아래는 테스트로 출력해 보기 위한 코드입니다.
print("결과 : " + hide_numbers('01033334444'));
```

    결과 : *******4444
