* while: 조건을 만족하지 않을 때 까지 수행합니다.


```python
n = 5
sum_ = 0
while n>1:
    sum_+=n
    n-=1
print(sum_)
```

    14


5부터 2까지만 더해지는 것을 볼 수 있습니다.


```python
def solution(n):
    answer = 0
    y = 0
    for i in range(17,-1,-1):
        if n//(3**i) >= 1: 
            y = i
            break
        
    thr = np.zeros(shape=(y+1,),dtype=np.int8) 
    
    for j in range(y, -1, -1):
        if n//(3**j) >=1 : 
            thr[j] = n//(3**j)
            n = n - (n//(3**j))*(3**j)
        elif n//(3**j) == 0 :
            thr[j] = 0
            
    for k in range(len(thr)) :
        answer += thr[k]*(3**(len(thr)-1-k))
    return answer
    
```
