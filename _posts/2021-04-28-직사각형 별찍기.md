* 사전지식
    * \n
* \n은 print문에서 다음줄로 넘어간다.


```python
print("가나다라마바" "\n" "아자차카")
```

    가나다라마바
    아자차카



```python
print(("a"*3 + "\n")*2)
```

    aaa
    aaa

* 나의 풀이


```python
def solution(n,m):
    for i in range(m):
        print('*'*n)

solution(5,3)
```
