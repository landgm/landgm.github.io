# 같은 숫자는 싫어

* 사전지식
    * continue

* continue 조건을 만족하는 경우 연산을 하지 않고 넘어갑니다


```python
for i in range(10):
    if i%2 ==0 : continue
    print(i)

```

* 짝수는 넘어가고 홀수만 출력하는 것을 볼 수 있습니다.


```python
def solution(arr):
    answer = []
    for i in range(len(arr)-1): 
        if arr[i] != arr[i+1]:
            answer.append(arr[i])
    
    answer.append(arr[-1])
    return answer
```


```python
a = [1,1,3,3,0,1,1]
```


```python
solution(a)
```




    [1, 3, 0, 1]

* 다른 사람 풀이


```python
def no_continuous(s):
    a = []
    for i in s:
        if a[-1:] == [i]: continue 
        a.append(i)
    return a

# 아래는 테스트로 출력해 보기 위한 코드입니다.
#print( no_continuous( "133303" ))
```
